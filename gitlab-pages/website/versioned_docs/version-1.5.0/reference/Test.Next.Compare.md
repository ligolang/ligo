<SyntaxTitle syntax="cameligo">

<Syntax syntax="cameligo">

The call `eq x y` returns `true` if, and only if, `x` and `y`
        are considered to be equal w.r.t. the order on the underlying
        type.

</Syntax>

        are considered to be equal w.r.t. the order on the underlying
        type.

</Syntax>
