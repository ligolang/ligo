import Syntax from '@theme/Syntax';

The predefined type `bool` has exactly two values: `true` and `false`.

<Syntax syntax="cameligo">

```cameligo group=booleans
let a : bool = true
let b : bool = false
```

</Syntax>

<Syntax syntax="jsligo">

```jsligo group=booleans
const a : bool = true;
const b : bool = false;
```

</Syntax>
